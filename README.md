

# 🧮 Calculadora De Física Em Python
Uma calculadora interativa no terminal que resolve problemas básicos de física, como energia cinética, velocidade e força.
<hr/>

## ⚡ Funcionalidades <br/>

 - Cálculo de Energia Cinética<br/>

 - Cálculo de Velocidade<br/>

 - Cálculo de Força<br/>

 - Validação de entradas do usuário<br/>

 - Aceita valores com ou sem unidades (ex: 10, 10kg, 10 m/s)<br/>

<hr/>

## 🧠 Fórmulas utilizadas<br/>
 - Energia Cinética <br/>
   E_c = (m * v²) / 2 <br/>
   
 - Velocidade <br/>
  v = d / t <br/>

 - Força<br/>
  F = m * a<br/>

## 💻 Exemplo de uso

=== CALCULADORA FÍSICA === <br/>
1 - Energia Cinética<br/>
2 - Velocidade<br/>
3 - Força<br/>
4 - Sair<br/>

Escolha uma opção: 1<br/>

Qual é o objeto? Bola <br/>
Qual a massa do(a) Bola em KG? 2kg <br/>
Qual a velocidade em m/s da Bola? 10 <br/>

A energia cinética do(a) Bola é 100.00J <br/>
